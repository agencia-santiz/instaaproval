# 🏗️ Skill: Implementation Workflow & Architecture

## 1. Project Architecture (Vanilla HTML/CSS/JS)

Para sites que precisam de máximo controle visual e performance:

```
project/
├── index.html
├── js/
│   ├── app.js              # Main entry, initializations
│   ├── scroll-animator.js  # Scroll-triggered animations
│   ├── smooth-scroll.js    # Smooth scroll engine  
│   └── cursor.js           # Custom cursor
├── css/
│   ├── reset.css           # CSS reset/normalize
│   ├── variables.css       # Design tokens
│   ├── typography.css      # Font imports & text styles
│   ├── animations.css      # Keyframes & animation classes
│   ├── layout.css          # Grid, containers, sections
│   └── components.css      # Buttons, cards, forms, navigation
├── assets/
│   ├── images/
│   ├── videos/
│   └── fonts/
└── favicon.ico
```
